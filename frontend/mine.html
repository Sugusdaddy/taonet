START MINING

Become a miner on the TaoNet network. Process AI inference tasks and earn rewards.

REQUIREMENTS

Token Holdings Any amount (more = higher tier)
Node.js v18 or higher
AI Model Local or API endpoint
Internet Stable connection
[1]

INSTALL THE SDK

Install the TaoNet miner SDK via npm:

npm install -g taonet-miner

Or clone directly from the repository:

git clone https://github.com/taonet/miner-sdk.git
cd miner-sdk
npm install
[2]

CONFIGURE YOUR MINER

Create a configuration file config.json:

{
  "wallet": "YOUR_SOLANA_WALLET_ADDRESS",
  "privateKey": "YOUR_PRIVATE_KEY",
  "endpoint": "http://204.168.139.31",
  "model": {
    "type": "openai",
    "apiKey": "YOUR_OPENAI_KEY",
    "model": "gpt-4"
  },
  "capabilities": ["text", "code"]
}
[3]

START MINING

Run the miner:

taonet-miner start --config config.json

Or with the cloned repository:

node index.js --config config.json

Your miner will connect to the network and start receiving tasks automatically.

CONFIGURATION OPTIONS

Option Description Required
wallet Your Solana wallet address Yes
privateKey Wallet private key for signing Yes
endpoint TaoNet API endpoint Yes
model.type Model provider (openai, anthropic, local) Yes
model.apiKey API key for the model provider Depends
capabilities Task types: text, code, image, trading Yes

TASK TYPES

text Text generation, Q&A, summarization
code Code generation, debugging, review
image Image generation, analysis
trading Market analysis, signals

REWARD STRUCTURE

Base Reward 10 tokens per task
Quality Bonus +5 tokens (score above 90)
Speed Bonus +3 tokens (under 2 seconds)
Streak Multiplier Up to 2x (30 day streak)
Staking Multiplier Up to 3x (Diamond tier)

Maximum reward per task: 10 * 2.0 * 3.0 = 60 tokens (with max multipliers)
