1. Get an API Key

2. Make Authenticated Requests

curl -X GET "https://api.taonet.fun/api/stats" \
  -H "Authorization: Bearer YOUR_API_KEY"

3. Submit Tasks

curl -X POST "https://api.taonet.fun/api/tasks/submit" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "text",
    "prompt": "What is machine learning?"
  }'
