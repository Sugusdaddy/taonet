OVERVIEW

TaoNet is a decentralized AI inference network. Submit tasks, miners process them, everyone earns.

Base URL

https://api.taonet.ai

Response Format

All responses are JSON. Successful responses include the requested data. Errors include an error field.

{
  "success": true,
  "data": { ... }
}

// Error
{
  "error": "Error message here"
}

AUTHENTICATION

Most endpoints require wallet signature authentication. Sign a challenge message with your wallet.

GET /api/auth/challenge/:address

Get a challenge message to sign.

// Response
{
  "message": "TaoNet Auth: 0x1234... - 1703275200",
  "timestamp": 1703275200
}
POST /api/auth/verify

Verify a signed message.

// Request
{
  "address": "0x1234...",
  "signature": "0xabcd...",
  "message": "TaoNet Auth: 0x1234... - 1703275200"
}

// Response
{
  "valid": true,
  "address": "0x1234..."
}

TASKS API

Task Types

TypeDescription
textText generation, Q&A, summarization
codeCode generation, debugging, review
tradingMarket analysis, price predictions
imageImage analysis, captioning
customAny other AI task
POST /api/tasks/submit

Submit a new task to the network.

// Request
{
  "type": "text",
  "prompt": "Explain quantum computing in simple terms",
  "params": {
    "maxTokens": 500,
    "temperature": 0.7
  },
  "requester": "0x1234...",
  "rewardPool": "1000000000000000000"  // 1 token in wei
}

// Response
{
  "success": true,
  "task": {
    "id": "65a1b2c3d4e5f6...",
    "type": "text",
    "status": "pending",
    "createdAt": "2024-01-15T10:30:00Z"
  }
}
GET /api/tasks/:taskId

Get task details.

GET /api/tasks/:taskId/result?timeout=30000

Wait for and get task result. Timeout in milliseconds (max 60000).

// Response (success)
{
  "success": true,
  "result": "Quantum computing uses quantum bits...",
  "miner": "0x5678...",
  "score": 95
}

// Response (pending)
{
  "success": false,
  "status": "pending",
  "message": "Task still processing"
}
GET /api/tasks?status=pending&type=text&limit=20

List tasks with filters.

WEBSOCKET API

Miners connect via WebSocket to receive tasks in real-time.

Connection

wss://api.taonet.ai/ws

Message Types

SEND auth

Authenticate after connecting.

{
  "type": "auth",
  "address": "0x1234...",
  "signature": "0xabcd..."  // Sign "TaoNet Miner Auth: 0x1234..."
}
RECEIVE new_task

Received when a task is assigned.

{
  "type": "new_task",
  "task": {
    "id": "65a1b2c3...",
    "type": "text",
    "prompt": "...",
    "params": { ... },
    "rewardPool": "1000000000000000000"
  }
}
SEND task_response

Submit task response.

{
  "type": "task_response",
  "address": "0x1234...",
  "payload": {
    "taskId": "65a1b2c3...",
    "response": { "text": "..." },
    "processingTime": 2500
  }
}
SEND heartbeat

Keep connection alive (send every 30s).

{ "type": "heartbeat", "address": "0x1234..." }

SDK & EXAMPLES

JavaScript/Node.js

const response = await fetch('https://api.taonet.ai/api/tasks/submit', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    type: 'text',
    prompt: 'What is the meaning of life?',
    rewardPool: '1000000000000000000'
  })
});

const { task } = await response.json();

// Wait for result
const result = await fetch(`https://api.taonet.ai/api/tasks/${task.id}/result?timeout=30000`)
  .then(r => r.json());

console.log(result.result);

Python

import requests

# Submit task
response = requests.post('https://api.taonet.ai/api/tasks/submit', json={
    'type': 'text',
    'prompt': 'What is the meaning of life?',
    'rewardPool': '1000000000000000000'
})

task_id = response.json()['task']['id']

# Get result
result = requests.get(f'https://api.taonet.ai/api/tasks/{task_id}/result?timeout=30000')
print(result.json()['result'])
