TaoNet Documentation

TaoNet is a decentralized AI inference network built on Solana. Miners process AI tasks and earn tokens for their contributions.

What is TaoNet?

TaoNet connects AI task requesters with distributed computing power. When you run a miner, you process AI inference requests and earn tokens.

Wallet Setup

TaoNet uses Solana wallets. We recommend Phantom for the best experience.

  1. Install Phantom Wallet
  2. Create or import a wallet
  3. Click "Connect" on TaoNet
  4. Approve the connection in Phantom

Browser Mining

The easiest way to start. Mine directly in your browser.

Note

Browser mining has rate limits. For higher earnings, use the CLI miner.

  1. Go to the Mine page
  2. Connect your wallet
  3. Register a miner name
  4. Click "Start Browser Mining"

CLI Miner

For serious mining. Higher rate limits and better rewards.

Installation
npm install -g taonet-miner
Configuration
taonet-miner config --wallet YOUR_WALLET_ADDRESS
Start Mining
taonet-miner start

Options

Flag Description
--wallet Your Solana wallet address
--provider AI provider (openai, anthropic, ollama)
--model Model to use for inference
--threads Number of parallel workers

How Earning Works

You earn tokens for each completed task. Your earnings depend on:

  • Base reward: 10 tokens per task
  • Quality bonus: +5 tokens for 80%+ quality score
  • Speed bonus: +3 tokens for responses under 30s
  • Tier multiplier: 1x to 3x based on token holdings
  • Streak bonus: +2% per consecutive day (max 20%)

Tier System

Hold more tokens to unlock higher earning multipliers.

Tier Tokens Required Multiplier
Bronze Any 1x
Silver 10,000 1.25x
Gold 100,000 1.5x
Platinum 1,000,000 2x
Diamond 10,000,000 3x

Jackpots

Random tasks trigger jackpot multipliers. Every task has a chance to hit.

Jackpot Multiplier Frequency
Mini 32x ~1 in 50
Regular 144x ~1 in 500
Mega 699x ~1 in 5,000
Ultra 5,163x ~1 in 50,000
